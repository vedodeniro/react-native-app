Cookie clicker app

En app där du klickar på en kaka och samlar cookies.

git clone https://github.com/vedodeniro/react-native-app.git

npm install

npx expo start --tunnel

React Native komponenter

- View
- Text
- Image
- Pressable
- TouchableOpacity
- Stylesheet

Expo SDK komponenter

- LinearGradient
- Haptics
- SecureStore
- expo Audio
